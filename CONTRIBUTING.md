# 🤝 Contributing to AI Studio 99-Page Generator

Thank you! This project welcomes contributions from TypeScript/Vite developers, Gemini API users, and AI content generation enthusiasts.

---

## 📋 How to Contribute

### 1. Fork & Clone
```bash
git clone https://github.com/SNTL84/ai-studio-99page-generator.git
cd ai-studio-99page-generator
```

### 2. Create a Feature Branch
```bash
git checkout -b feature/your-feature-name
```

### 3. Install & Run
```bash
npm install
npm run dev
```

### 4. Commit & Push
```bash
git commit -m "feat: describe your change"
git push origin feature/your-feature-name
```

### 5. Open a Pull Request
