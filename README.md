## 🎯 What Is This?

A **Vite + TypeScript** web app powered by **Google Gemini API** that generates complete 99-page content structures from just 3 prompts. Ideal for developers building AI-assisted content pipelines.

## 🚀 Quick Start

**Live Demo:** [Click here →](https://sntl84.github.io/ai-studio-99page-generator/)

```bash
git clone https://github.com/SNTL84/ai-studio-99page-generator.git
cd ai-studio-99page-generator
npm install
# Add your Gemini API key in .env
echo "VITE_GEMINI_API_KEY=your_key_here" > .env
npm run dev
```

Get a free Gemini API key at [ai.google.dev](https://ai.google.dev/)
