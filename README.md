## 🚀 Run Locally

```bash
# Clone
git clone https://github.com/SNTL84/ai-studio-99page-generator
cd ai-studio-99page-generator

# Install
npm install

# Add Gemini API key
cp .env.example .env.local
# Set GEMINI_API_KEY=your_key_here

# Run
npm run dev
```
