## ✨ What It Does

- Generates structured multi-page web applications via AI prompting
- Uses Gemini API for intelligent content and layout generation
- TypeScript + Vite stack for production-grade output
- Clean modular architecture: `assets`, `data`, `scripts`, `services`, `src`

---

## 🛠️ Run Locally

**Prerequisites:** Node.js 18+

```bash
# 1. Clone the repo
git clone https://github.com/SNTL84/ai-studio-99page-generator

# 2. Install dependencies
cd ai-studio-99page-generator
npm install

# 3. Add your Gemini API key
cp .env.example .env.local
# Edit .env.local: set GEMINI_API_KEY=your_key_here

# 4. Start dev server
npm run dev
```
